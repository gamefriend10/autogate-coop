# AutoGate

## Adding a battle group

1. Create a base core and inherit from this. This controls the name for all.
2. Create snow tag for battle group.
3. Create spawn function.
4. Create shop>hand morph and shop core. Use morph ability in shop core and add tags.
5. Create hand>staging construct and hand core. Use construct ability in hand core and add tags. Set build cost.
6. Create staging core. Add sell ability. Set build cost and footprint.
